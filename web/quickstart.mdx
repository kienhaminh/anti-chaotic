## Setup your development

Learn how to update your local development environment to work with Anti-Chaotic.

### Installation

<CodeGroup>

```bash npm
npm install -g anti-chaotic
```

```bash yarn
yarn global add anti-chaotic
```

</CodeGroup>

## Usage

Once installed, you can run the `anti-chaotic` command to start the agentic process.

```bash
anti-chaotic start
```
